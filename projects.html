Online Judge

The project was aimed at building the backend of a web based Judge which checks a code for Correctness, Run time and memory. Haskell is the ideal language for this since we can effectively use Concurrent threads and memory to lazily compute everything. Further this can be deployed using web based framework such as Flask which is based on Python. View the code at Github.com

Autonomous Underwater Vehicle

The project is aimed at creating an Autonomous Robot which runs underwater. The programming problem can be divided into 3 main sections which are Controls, Image Processing and Aritificial Intelligence algorithms. Presently we use Robot OS' effective message passing, goal callback and effective unit testing techniques to optimize the controls and seperating the logical levels from the actuation levels. Further we use OpenCV for image processing.
View the code at Github.com

TicTacToe-AI

The basic idea is to effectively use the laziness of Haskell to easily find the winning strategy in a given game of TicTacToe without overflowing the stacks. Lazy computation and the fact that Haskell runs similar computations only once.

Grammar Checker

The project is aimed at building a Haskell based code which defines a basic parseable type system that takes input a Grammar which checks if the Grammar is Context Free and further if it is LL(1). If it is then it further would aim to check if it can be used to form a programming language which is Turing Complete. View the code at Github.com
